# User Manual
### 0. Install
<div>
  <li>사용하고 있는 브라우저의 확장프로그램에서 Chrome 웹스토어에 접속합니다.</li>
  <li>검색창에 'unist blackboard'를 검색해서 "Blackboard Extension"을 찾아서 설치합니다. (다른 확장 프로그램과 헷갈리지 않도록 주의하세요)</li>
